## [1.0.0] - 2026-04-03

### Added
- 출석 체크 기능 (하루 1회, 중복 출석 방지)
- 기본 포인트 지급 (설정 가능)
- 순위 보너스 포인트 (1등/2등/3등)
- 개근 보너스 포인트 (주간/월간/연간)
- 랜덤 추가 포인트 (확률 및 범위 설정 가능)
- 출석 시간 제한 설정
- 자동 출석 기능 (로그인 시 자동 출석 처리)
- 기본 인삿말 목록 관리 (랜덤 인삿말 제공)
- 유저 출석부 화면 (출석 달력, 개근 현황, 오늘의 출석 목록)
- 관리자 설정 화면 (보너스 설정, 시간 제한, 랜덤 포인트, 인삿말 관리)
- 관리자 스킨 관리 화면
- 관리자 출석 통계 API
- 프론트엔드 실시간 시각 표시 (JS 핸들러)
- 다국어 지원 (한국어, 영어)
- 백엔드 테스트 (Unit/Feature)
- 프론트엔드 레이아웃 렌더링 테스트 (Vitest)
- GitHub Actions CI 워크플로우
