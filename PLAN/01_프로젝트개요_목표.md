# 01. 프로젝트 개요 및 목표

## 1) 프로젝트 배경

- 본 저장소는 그누보드7용 출석부 모듈(`yjsoft-attendance`)을 독립 확장 모듈로 제공하기 위한 목적을 가진다.
- 코어 수정 최소화 원칙을 지키며, 모듈/템플릿/권한 체계와 충돌 없이 동작해야 한다.
- 초기 릴리스 버전은 `1.0.0`으로 시작한다.

## 2) 목표

- 사용자 친화적인 출석 경험 제공
  - 출석 체크 + 인삿말 작성
  - 출석/결석 날짜 시각화
  - 실시간 현재 시각 표시
- 운영자 친화적인 정책 제어
  - 자동출석 on/off
  - 출석 가능 시간 설정
  - 랜덤 포인트 부여 정책
- 포인트 동기부여 강화
  - 연속 출석(주/월/년) 보너스
  - 일일 1~3등 선착순 보너스

## 3) 범위 (In Scope)

- 출석 데이터 기록/조회 API
- 출석부 사용자 화면(User)
- 관리자 설정 화면(Admin)
- 포인트 계산 로직 및 로그
- 다국어 키 기반 문구 관리

## 4) 범위 제외 (Out of Scope, v1.0.0)

- 출석 랭킹 시즌제(월간 리셋 보상 등)
- 푸시/알림톡 연동
- 외부 포인트 시스템 연동
- 게임화 미션/배지 시스템

## 5) 성공 기준 (KPI/DoD)

- 기능 요구사항 100% 충족
- 금지사항 위반 0건
- 주요 API/서비스 테스트 통과
- 동시성 상황(중복 출석)에서 데이터 정합성 보장
- 운영 설정 변경 시 즉시 정책 반영
